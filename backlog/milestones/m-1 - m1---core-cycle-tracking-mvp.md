## Description

Status: in progress (current milestone). Ship the smallest version that already solves the main user problem. Features: onboarding, cycle/period length setup, calendar view, log period start/end, flow intensity, predicted next period, current cycle day, daily check-in (mood, energy, pain, discharge, sleep quality, optional note), symptom logging with predefined tags (cramps, headache, nausea, acne, bloating, breast tenderness, fatigue, PMS). See product-roadmap.md "Milestone 1: Core Cycle Tracking MVP".
