## Description

Status: open. Make the app more useful during healthcare conversations and longer-term tracking. Features: export to PDF, export to CSV, doctor-friendly summary view, date range filters, symptom/cycle period filters, clearer trend visuals. Exports should include cycle dates, period logs, symptom logs, check-in summaries, and notes where the user opts in. See product-roadmap.md "Milestone 3: Export and Better Review".
